# Creating Alerts

EchoGTFS allows you to create and manage GTFS-RT ServiceAlerts easily via the web interface.

## Steps to Create an Alert
1. Log in with your credentials.
2. Navigate to the **Alerts** section.
3. Click **Capture**.
4. Set alert severity, effect and cause.
5. Define the active period for the alert.
6. Select affected routes, stops, or trips from your GTFS feed.
7. Add descriptions in multiple languages.
8. Publish the alert to make it available via the GTFS-RT feed.

Additionally, you can enable and disable each alert. Disabled alerts are not published using the GTFS-RT feed.

You also can [configure external data sources](data_sources.md) to import alerts from external sources, for example SIRI-SX. _Note: Alerts imported from external sources can only be viewed, but there's no way to edit the alerts. External alerts can only be disabled to suppress them in the GTFS-RT feed._
