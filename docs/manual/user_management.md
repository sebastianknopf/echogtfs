# User Management

EchoGTFS provides a user management system to control access to the platform. This allows administrators to add, edit, and remove users, as well as assign roles.

## Accessing User Management
- Log in as an administrator.
- Navigate to the **Accounts** section in the web interface.

## User Roles
- **Admin:** Full access to all features, including user management, alert creation, data source and system configuration.
- **Poweruser:** Can create and manage alerts, manage data source configurations, but cannot manage other users or system settings.
- **User:** Can create and manage alerts, but cannot manage data sources, users or system settings.

## Managing Users
- **Add User:** Click the "Add User" button, fill in the required details (username, password, role), and save.
- **Edit User:** Select a user from the list and update their information or role.
- **Delete User:** Remove a user from the system. (Admins cannot delete themselves.)

## Security Notes
- Always use strong, unique passwords for all accounts.
- Only assign admin rights to trusted personnel.
